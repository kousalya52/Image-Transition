# 🖼️ Image Transition Slider

A responsive **Before & After Image Comparison Slider** built using **HTML, CSS, and JavaScript**. Users can move the mouse across the image to reveal and compare two versions of an image interactively.

## 🌐 Live Demo

## 🚀 Features

* Interactive image comparison effect
* Smooth before-and-after image transition
* Mouse-controlled slider
* Lightweight and fast
* Responsive design
* Pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ⚙️ How It Works

1. The background image is displayed inside the container.
2. The original image is placed on top with a limited width.
3. Moving the mouse across the image changes the width of the overlay image.
4. A vertical slider line follows the mouse position.
5. This creates a before-and-after comparison effect.
